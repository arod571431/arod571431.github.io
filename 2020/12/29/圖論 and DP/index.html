DP方法

由於只需要找矩形而不是正方形
所以
我們可以切成看每條邊然後向下看
以範測為例

0 0 1 1 0
則可以看成 0 0 1 1 0
0 0 1 1 0
0 1 1 1 1
則可以看成 0 1 2 2 1
0 0 1 1 0
0 1 1 1 1
0 1 1 1 1
則可以看成 0 2 3 3 2
0 0 1 1 0
0 1 1 1 1
0 1 1 1 1
0 0 1 0 0
則可以看成 0 0 4 0 0

也就是說每次看單條邊時
我們可以將跟他上面已有的長度+1
如果遇到0則淨空
然後每次算完再做矩形面積計算
記得每次計算取不是零的最小值算
然後再取計算結果的最大值
